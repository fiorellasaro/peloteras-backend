# Peloteras Backend

## Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
```
yarn install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```

7. Ejecutar la aplicacion en dev: 
```
yarn start:dev
```

## Stack usado
* Nest
